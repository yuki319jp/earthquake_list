# UIデザインについて
- Apple Human Interface Guidelinesをベースにしてください
 - document: https://developer.apple.com/jp/design/human-interface-guidelines
- フォントはGoogle Fonts , アイコンなどはLucideを使用してください。
  - Noto Sansを採用してください

# アニメーションについて
- Tailwind CSSを使用してください
- あまり多用せず、入れると効果的だと判断したもののみに追加してください

# カラー
- モノトーンを採用してください
